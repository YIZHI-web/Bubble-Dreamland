## 活动名称：《勇者试炼·千层回廊》

### 一、活动概述

- **所属游戏类型**：横版动作Roguelite手游（自拟游戏名《影境》）
- **活动类型**：**连续闯关**（线性层级递增，不可重复挑战同一关）
- **活动周期**：14天（每赛季重置）
- **参与条件**：玩家等级≥20级，解锁“秘境”系统
- **活动入口**：主界面-活动中心-【勇者试炼】图标

### 二、活动背景设定

> “传说在帝国边境深处，有一座无尽的千层回廊，每一层都封印着强大的守关者。唯有真正的勇者才能连续突破重重考验，抵达深层获得先贤遗物。”  
> 本次活动开启前50层，后续版本逐步开放更高层。

### 三、核心规则

#### 1. 闯关结构
- 共 **50层**（也可设计为100层，根据工期调整）。  
- 每一层为一个独立的小型房间，包含：
  - 普通层：若干杂兵（3-6只） + 陷阱或环境机关
  - 精英层（每5层）：1只精英怪 + 2-3只杂兵
  - 首领层（每10层）：1只BOSS，拥有特殊机制
- **连线规则**：必须从第1层开始，依次挑战2、3……直到50层。**不可跳关，不可重复挑战已通过的层。**

#### 2. 挑战模式
- **单人挑战**：默认模式，只能使用玩家自己的角色。
- **协同模式（可选）**：允许2人联机，但怪物生命值提升150%，攻击力提升30%，且每名玩家只能携带1名角色（共2名角色出战）。

#### 3. 生命与状态继承机制
- **生命继承**：每层结束后，角色的当前生命值、元素能量/怒气值、技能冷却时间**保留**到下一层。
- **无法中途回血**：除了通关后出现的回血祭坛（每5层固定一个回满血祭坛）、或使用自带回复道具（限制每3层可使用1次血瓶）。
- **角色死亡**：若角色在某一层死亡，则**整次闯关失败**，必须从第1层重新开始（记录最高已通关层数，可用于领取奖励）。

#### 4. 动态难度调整
- 根据玩家 **当前战力评分（装备+角色等级+技能等级综合）** 自动调整怪物属性：
  - 战力1万以下 → 怪物等级30，血量100%
  - 战力1-2万 → 怪物等级45，血量120%
  - 战力2万以上 → 怪物等级60，血量150%，并增加1-2种额外技能。
- **注**：战力区间每两周校准一次，防止玩家压战力逃课。

#### 5. 中途保存与次数限制
- 每日免费挑战次数：**3次**（消耗完毕可花费50钻石购买1次，每日最多额外购买2次）。
- **自动保存**：每突破5层（第5、10、15…层）自动保存一个“进度点”。玩家下次挑战时可以选择：
  - 从上次的进度点继续（例如上次打到23层，自动保存点在20层，则从20层开始）
  - 或者彻底重头开始。
- 若从进度点继续，该进度点之后已通关的层数不会再给已领取的首通奖励。

### 四、特色规则

#### 1. 随机增益“勇者祝福”
- 每通过**3层**（即第4层开始前、第7层开始前…），会出现一个三选一的随机祝福。
- 祝福效果举例：
  - **狂暴之心**：攻击力+15%，但受到的伤害+10%
  - **元素共鸣**：元素充能效率+30%
  - **生命护佑**：每层开始获得5%最大生命值的护盾
  - **财富之握**：该次闯关中所有金币掉落+50%
  - **时间停滞**：受到致命伤害时，强制保留1点生命并进入2秒无敌（整局仅触发1次）
- 祝福可叠加（同类型可多次选择，效果加法叠加），直到闯关结束或死亡时消失。

#### 2. 特殊层“试炼之间”
- 在第8、18、28、38、48层设置特殊关卡，不战斗，而是**趣味挑战**：
  - 躲避弹幕（生存30秒）
  - 限时击打机关（要求一定攻速）
  - 记忆匹配（翻牌）
- 完成特殊层可获得额外奖券，且不消耗生命值。

#### 3. 元素环境变化
- 每10层更换一种环境主题，影响玩法：
  - 1-10层：**烈焰回廊** – 地面周期性喷发火焰，对所有单位造成火元素伤害。
  - 11-20层：**冰霜回廊** – 移动速度降低20%，技能冷却延长15%。
  - 21-30层：**雷霆回廊** – 场地中出现会移动的雷球，碰到后麻痹1秒。
  - 31-40层：**暗影回廊** – 视野缩小，只有角色周围光圈可见。
  - 41-50层：**混沌回廊** – 每15秒随机切换一种上述环境效果。

#### 4. 排行榜与重置
- **首通排行榜**：按全服首次达到50层的时间排序，前100名获得限定称号+头像框。
- **速通排行榜**：按从第1层到通关（50层）的总用时排序（暂停不计时），每周一结算发奖。
- 每赛季（14天）结束后，清空所有进度和祝福，玩家必须重新从第1层开始挑战。已获得的奖励不会回收。

### 六、活动UI与流程

```
[入口] → 选择“开始挑战”（或“从进度点继续”） → 确认出战角色（单人/联机） → 
加载第N层 → 战斗 → 胜利 → 若为3的倍数层则弹出三选一祝福 → 进入下一层 → 
若失败则弹出结算界面：展示最高层数、获得奖励、是否使用钻石复活？（限制：每天最多复活2次，每次100钻，复活后满血满能量但不清除祝福）
```

### 七、数值平衡参考

- **每层怪物强度增长曲线**：  
  基础HP = 1000 * (1.08^层数) ，每10层额外乘以1.2。  
  攻击力 = 100 * (1.05^层数) 。  
  例如第50层BOSS血量 ≈ 1000 * 1.08^50 ≈ 1000 * 46.9 ≈ 46900，再乘以1.2^5≈2.5 → 约117万（符合高难挑战）。

- **通关时间预估**：  
  普通层30秒~1分钟，精英层1~2分钟，BOSS层2~3分钟。  
  完整50层理想总时长约1.5~2小时（适合硬核玩家分次挑战）。

### 八、活动可行性及预期效果

- **对玩家的价值**：
  - 提供中期挑战目标，解决长草期无聊问题。
  - 连续闯关机制强化“步步为营”的紧张感，随机祝福增加重复可玩性。
  - 丰厚外观奖励刺激社交传播。

- **对游戏KPI的影响**：
  - 每日挑战次数限制及复活消耗，促进钻石消耗。
  - 首通竞争和速通榜刺激活跃度与在线时长。
  - 联机模式促进好友互动与邀请拉新。

- **技术实现难度**：  
  中等。需要保存每一层的怪物配置、祝福组合、玩家状态（血量/能量）。现有Roguelite框架基本可复用。
